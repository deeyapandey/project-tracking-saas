# Project Tracking System

A **full-stack web application** designed to help teams manage projects, tasks, and deadlines efficiently.  
This system allows team members to create projects, assign tasks, track progress, and stay organized.

---

## Features

- **User Management**: Sign up, login, and role-based access for team members.
- **Project Management**: Create, update, and delete projects with deadlines and descriptions.
- **Task Management**: Assign tasks to team members, set priorities, and track status.
- **Progress Tracking**: View project progress and task completion at a glance.
- **Notifications**: Get alerts for upcoming deadlines or task updates.

---

## Tech Stack

- **Backend**:

  - .NET 10
  - Entity Framework Core
  - SQL Server
  - AutoMapper for DTOs
  - REST APIs with proper error handling

- **Frontend**:
  - React
  - Tailwind CSS for styling
  - React Hook Form for forms
  - Axios / API client for backend communication
